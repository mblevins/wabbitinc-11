# Wabbit Inc

Wabbitinc as an eleventy site. Shamelessly stolen from https://github.com/11ty/eleventy-base-blog.git .

## To test locally

```
npx @11ty/eleventy --serve
```
use npm for package updates, and nvm for node/npm updates
